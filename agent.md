# Agent 작업 지침

1. 전부 구현해라.
2. 작업이나 단계를 완료하면 계획 문서에서 완료로 표시해라.
3. 모든 작업과 단계가 완료될 때까지 멈추지 마라.
4. `any` 나 `unknown` 타입을 쓰지 말고, 지속적으로 typecheck 를 실행해서 새로운 문제를 만들지 마라.
5. 아키텍처 변경이나 워크플로우 변경 시 `research.md` 에 내역을 업데이트해라.
6. 마지막에는 Git 에 커밋하고 `timeline.md` 문서에 Git 에 커밋한 시간과 작업 내역을 요약해서 저장해라.

예시:

- `[2026-04-23-11-38] 자막 기능 업데이트`
- `[2026-04-23-11-38] 썸네일 기능 업데이트`
